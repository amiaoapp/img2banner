# Image to banner

本工具用于把上传或粘贴的（正方形）图片自动放到可配置的画布中心并导出为透明背景的 PNG / WebP。

## 使用方法：
- 打开 `index.html`（在本地双击或用静态服务器）。
- 拖拽、点击上传或按 Cmd+V 粘贴图片（支持 GIF，取首帧）。
- 在左侧配置画布大小、内图尺寸、背景、输出格式，右侧预览实时更新。
- 点击“下载”导出，或“复制为 PNG”将图片复制到剪贴板。

说明：导出为 WebP 会保留透明（若浏览器支持）。GIF 只会使用首帧作为静态图片。
